# Study Planners 

A web-based study and assignment planner with Kanban board and syllabus tracking.

## Features
- User login system
- Syllabus tracker with progress bar
- Kanban board (To Do / In Progress / Done)
- Assignment deadline tracking
- Dashboard analytics

## Tech Stack
- HTML, CSS, Bootstrap
- JavaScript (DOM, AJAX)
- PHP
- MySQL (XAMPP)

## How to Run
1. Start XAMPP (Apache + MySQL)
2. Import database.sql into phpMyAdmin
3. Open: http://localhost/WD1%20project/dashboard.php
